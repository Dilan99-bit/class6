# Semana de Lectura Creativa

## Descripción

Este proyecto es una página web creada mientras estaba aprendiendo a trabajar con **HTML**, **CSS** .

## Contenido

- **index.html** - Página principal del proyecto
- **css/** - Estilos y archivos de CSS
- **img/** - Imágenes utilizadas en la página
- **j.s/** - Scripts de JavaScript

## Propósito

Este proyecto fue desarrollado como parte del proceso de aprendizaje de desarrollo web, practicando:

- Estructura de documentos HTML
- Diseño y estilos con CSS
- Interactividad con JavaScript
- Organización de archivos y carpetas

## Autor

Desarrollado mientras se aprendía HTML y desarrollo web.
